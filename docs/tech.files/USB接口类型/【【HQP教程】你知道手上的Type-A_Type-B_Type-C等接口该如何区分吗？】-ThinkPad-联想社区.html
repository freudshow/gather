4 天前 上传
比较常见的USB接口分为：Type-A、Type-B和Type-C
Type-A
这个接口大家都很熟悉啦，就是电脑上，充电器上用的。
优点：普及率高。
缺点：需要区分正反、接口太大。

Mini USB
MINI USB前几年在手机、MP3上使用非常广泛，又被称做Mini USB口。目前已经很难见到了。
Type-B
打印机上常用的口，平时不怎么常见。

Type-C
Type-C就是这段时间非常火热的接口了，用来取代Type-B和Micro USB的。
Type的优点：不分正反都可插入、尺寸变小。
缺点：普及率较低，无法和Micro USB兼容。

Micro USB
目前绝大部分Android手机均采用Micro USB接口规范。
优点:普及率高。
缺点：需要区分正反。接口耐用度不高。

Thunderbolt 3
Thunderbolt 3将与Type-C接口兼容，变得更加普及。
目前ThinkPad P70上搭载两个Thunderbolt 3接口。
------------------------------------------USB传输标准-------------------------------------------
USB：
USB接口最早有USB1.0、USB1.1，外观和Type-A接口一样。由于太老且不容易见到，所以我们不讨论了。
目前比较常用的有：
USB2.0
传输速率：480Mbps（60MB/s）
供电：5V/0.5A

USB3.0
传输速率：5Gbps（625MB/s）
供电：5V/0.9A

USB3.1
传输速率：10Gbps(1250MB/S)
供电：20V/5A

Thunderbolt2接口


Thunderbolt的研发初衷是为了替代并且统一目前电脑上数量繁多性能参差不齐的扩展接口，由Intel负责研发，Apple负责推广，但是后来除了Apple自家产品，其他产品使用Thunderbolt接口的设备少之又少，并且和无法兼容USB标准兼容。很大程度上影响了Thunderbolt的推广。
后来，因为Thunderbolt和USB同样是数据端口，英特尔宣布将改变Thunderbolt 3的数据接口形式，和Type-C统一接口并且兼容USB3.1标准，不过仍然保留闪电logo。


问答环节：
1、那么既然现在都是Type-C了，为什么还有Thunderbolt、USB之分呢？
答：其实目前的Type-C只是一种接口的规范，和Thunderbolt、USB并没有关系。就好比USB接口也有2.0/3.0之分一样，
去年年初，Nokia发布N1平板，采用Type-C的接口，但是却是2.0总线，这就意味着，哪怕你用3.1的Type-C和N1互联，传输速度也只能是2.0的速度。
比如HQP的Nexus 6P，虽然也使用上了Type-C接口，但是确实2.0的总线。据HQP了解，只有Lenovo旗下的ZUK，搭载的Type-C使用了USB3.0总线。
具体Type-C设备的传输速率如何，需要看里边的总线。


2、我买设备需要是否考虑Type-C接口吗？
答：目前的Type-C已经逐渐普及，如果你手边有Type-C设备，那么就推荐购买。如果没有或者不是Geek，爱吃螃蟹。还是老老实实的用Type-A和Micro USB吧。

3、Thunderbolt 3是否支持USB3.1呢？
答：英特尔给出的官方文档表示将会兼容USB3.1标准，但是具体是否支持需要看各大OEM厂商。目前网上有说支持，有说不支持的。大家购买的时候最好具体咨询OEM厂商。
